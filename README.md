**console.json()**

*Proof of concept for JSON syntax highlighting in the browser console. Great for debugging :)*

To use simply include in your project and type into the console:

<pre>
console.json({
    a: 'hello world',
    b: {
        key: 'value'
    },
    c: [3, 2, 1],
    d: 1,
    e: true,
    f: undefined,
    g: null
})
</pre>
